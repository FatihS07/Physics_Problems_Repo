# Task 09 – Schwarzschild Radius

## Problem Statement

1. Calculate $r_s$ for the Sun and Earth.
2. Calculate redshift at $r = 1.5r_s$.
3. Analyze the limit $r \to r_s$.
4. Interpret the event horizon.

## Theory

The Schwarzschild radius is:

$$
r_s = \frac{2GM}{c^2}
$$

The gravitational redshift factor is $z = \frac{1}{\sqrt{1 - r_s/r}} - 1$.

## Step-by-Step Solution

### 1. Calculations

- Sun ($2 \times 10^{30}\,\text{kg}$): $r_s \approx 3\,\text{km}$.
- Earth ($6 \times 10^{24}\,\text{kg}$): $r_s \approx 9\,\text{mm}$.

### 2. Redshift at $1.5r_s$

$$
1 + z = \frac{1}{\sqrt{1 - \frac{1}{1.5}}} = \frac{1}{\sqrt{1/3}} = \sqrt{3} \approx 1.732
$$

## Interpretation

As $r \to r_s$, the redshift becomes infinite. This means light can no longer escape the region, defining the event horizon of a black hole.
